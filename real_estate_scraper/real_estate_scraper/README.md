# Real Estate Web Scraper

A Scrapy-based web scraper that extracts real estate listings data from homes-jordan.com.

## Description

This project crawls property listings and extracts structured data including:

- Listing type (Rent / Sale)
- Price (Monthly / Annual / Sale)
- Bedrooms
- Bathrooms
- Area (sqm)
- Furnished status
- Floor information
- Location
- Description
- Special features (Pool, Furnished, Floor, etc.)

The scraper handles:
- Different price formats
- Studio apartments (ST)
- Missing numeric values
- Data cleaning and normalization

---

## Technologies Used

- Python 3
- Scrapy
- XPath
- Regular Expressions
- Virtual Environment (venv)
